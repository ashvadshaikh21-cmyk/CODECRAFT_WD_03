 CODECRAFT_WD_03
 Tic Tac Toe Web Application

Overview
This is an interactive Tic Tac Toe web application built using HTML, CSS, and JavaScript.  
The game supports both 2-player mode and an AI opponent.

Live Demo:
https://ashvadshaikh21-cmyk.github.io/CODECRAFT_WD_03/

Features
- 2 Player Mode
- Play vs Unbeatable AI (Minimax Algorithm)
- Score Counter (X Wins, O Wins, Draws)
- Winning Cells Highlighted
- Responsive Design (Mobile Friendly)
- Restart Game Option

 Technologies Used
- HTML5
- CSS3 (Grid + Responsive Design)
- JavaScript (DOM Manipulation + Minimax AI)

AI Logic
The AI uses the Minimax algorithm to evaluate all possible outcomes and always plays the optimal move, making it unbeatable.
